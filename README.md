# jpm-coffee-inventory

A Basic Spring JPA app with an in-memory H2 DB

## Prerequisites
You will need to following tools in order to work with this project and code

- git (http://git-scm.com/)
- JDK 19+ (http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html)
- Maven 3.x+ (http://maven.apache.org/)
- An IDE of your choice. (Eclipse, IntelliJ, Spring STS, Netbeans, etc.)

## Getting Started
To run this project locally, perform the following steps.

- Clone project to your machine using git - "git clone https://github.com/mustafah17/jpm-coffee-inventory.git"
- Import the project into your IDE using the maven pom.xml.
- Run the JUnit tests in the src/test/java folder. If all pass you are good to go.

Swagger: run the CoffeeShopApplication then go to http://localhost:8080/swagger-ui/index.html
